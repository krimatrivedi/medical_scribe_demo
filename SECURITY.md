# Security Policy

## Supported Versions

Currently, only the latest version of **Medical Scribe AI** is supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a top priority for this project, especially given its clinical nature. If you discover a security vulnerability, please report it via the following process:

1.  **Do not** open a public issue.
2.  Send an email to the project maintainer (see profile).
3.  Include a detailed description of the vulnerability and steps to reproduce.

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary.

## Privacy Note
This is a demonstration project. Do not use real Patient Health Information (PHI) or Personally Identifiable Information (PII) during testing. This application is not yet HIPAA compliant.
